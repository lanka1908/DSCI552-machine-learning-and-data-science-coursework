# DSCI 552 – Machine Learning & Data Science Coursework

Collection of assignments from DSCI 552 (Machine Learning & Data Science) at USC, 
taught by Professor Mohammad Reza Rajati.

## Notebooks

| File | Topic |
|------|-------|
| HW1 | K-Nearest Neighbors – Vertebral Column dataset (Euclidean, Manhattan, Minkowski distances) |
| HW2 | Linear & Multiple Regression – Combined Cycle Power Plant energy output prediction |
| HW3 | Time Series Feature Extraction – AReM human activity recognition dataset |
| HW4 | Binary & Multi-class Classification – Logistic regression on AReM activity data |
| HW5 | Decision Trees & Regularized Regression – LASSO/Ridge on community crime data |
| HW7 | Multi-label SVM & K-Means Clustering – Anuran frog call classification |

Python · scikit-learn · Pandas · NumPy · Matplotlib · Seaborn · statsmodels
